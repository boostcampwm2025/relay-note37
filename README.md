## 이건 첫 번째 LESSON

![hello](./hello.jpg)

`입터졌GPT`는 AI의 **창의성을 활용하여 커뮤니티 멤버 간의 자연스러운 소통을 유도**하고 AI가 각자의 상황에 맞춘 **감성적, 사교적 템플릿**을 제안하여 **대화의 시작을 쉽게 하고, 긍정적인 팀 분위기**와 **더욱 친밀하고 유대감 있는 관계를 형성** 하는 것을 목표로 합니다.
